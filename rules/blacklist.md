# 禁用词库（Jargon Blacklist）

以下词汇绝对禁止出现在面向 C 端用户的文案中。遇到时必须替换为右侧的"人话"版本。

| 禁用词 | 替换为 |
|---|---|
| 部署 | 启用 / 安装 / 开始使用 |
| API | 接口（或直接省略） |
| Token | 使用额度 / 消耗量 |
| 环境配置 | 前期设置 |
| 挂载 | 连接 / 加载 |
| 沙箱 | 隔离的安全环境 |
| MCP | 插件协议（或直接省略） |
| 代码投毒 | 恶意代码植入 |
| 依赖 | 底层组件 |
| 容器/镜像 | 运行环境 |
| 跑通 | 顺利完成 |
| 报错 | 技术错误 / 异常 |
| 炼丹 | 反复调试 |
| 开源模型 | 免费开放的 AI 模型 |
